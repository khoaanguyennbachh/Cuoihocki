#Tây Du Ký – Hành Trình Thỉnh Kinh

Tây Du Ký – Hành Trình Thỉnh Kinh is a digital web project inspired by the legendary Journey to the West. Explore mythical characters, iconic challenges, divine items, and interactive storytelling in an immersive, beautifully themed online experience.

## Overview 

The Tây Du Ký web project is a creative, front-end focused website that retells the sacred journey to retrieve Buddhist scriptures through character showcases, animated slides, and themed sections based on the 81 trials.

---

## Objectives:

Recreate the Journey to the West in a dynamic digital format

Provide an interactive, mythology-inspired browsing experience

Demonstrate web design and front-end development skills

Highlight cultural storytelling through modern web technology

---

## Features

Responsive layout compatible with desktop, tablet, and mobile

Automatic slideshow presenting key moments from the journey

Character gallery featuring Xuanzang, Sun Wukong, Zhu Bajie, Sha Wujing, White Dragon Horse, and Guanyin

Highlights of famous challenges such as Fire Mountain, White Bone Spirit, and the Six-Eared Macaque

Themed UI design with classic red, gold, and traditional Eastern style

Interactive celestial contact form to “request divine assistance”
